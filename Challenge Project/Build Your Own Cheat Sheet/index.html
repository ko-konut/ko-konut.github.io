HTML part

Some Concepts

  1. HTTP (Hypertext transfer protocol) is a standartized set of rules for how browsers and servers tranfer data to one another.

  2. Examples of HTTP statuses:

    Number of status Explanation
    200 The request has succeeded
    301 The resource has been moved and the client is being redirected
    404 The requested resource was not found
    505 The server encountered an unexpected error

  3. Way of working:
    The client makes the request and browser sends it to server → Server send HTTP which is contains CSS with styles → Browser request CSS → Server send it → Browser apply CSS styles to the webpage
